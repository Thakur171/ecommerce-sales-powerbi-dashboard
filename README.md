# 📊 E-Commerce Sales Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive Power BI dashboard analyzing e-commerce sales data. It provides insights into revenue, customer behavior, product performance, and regional trends.

---

## 📂 Dataset

The dataset includes:

* Customers data
* Orders data
* Products data

---

## 📊 Key KPIs

* Total Revenue: ₹1.25bn
* Average Order Value (AOV): ₹112.8K
* Total Orders: 15.7K
* Cancellation Rate: 29.72%
* Lost Revenue due to Cancellation: ₹525.4M

---

## 📈 Key Insights

* Highest revenue comes from **Laptop and Mobile category**
* Maharashtra and Gujarat contribute the most sales
* Peak revenue observed in early quarters of 2024
* Significant revenue loss due to high cancellation rate

---

## 🧠 Data Model

* Orders table connected with Customers and Products using CustomerID and ProductID
* Star schema used for efficient analysis

---

## 🛠 Tools Used

* Power BI
* Data Modeling
* DAX (for KPI calculations)
